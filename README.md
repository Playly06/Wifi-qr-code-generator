# Wifi-qr-code-generator
It is an app that generates you a qr code for your wifi so you can connect to it.

This is a guide if you struggle for the security type:
If you have no idea what is this about, follow these steps:
1-Go to settings
2-Go to Network and Internet
3-Go to Wifi
4-Click to your wifi
5-Scroll down until you see "security type"
5.a-if you see WPA2 or WP3-Personnal,then select "WPA" on the app
5.b-if there is no password in your wifi then select "nopass" on the app
